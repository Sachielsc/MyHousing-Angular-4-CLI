Tips about angular-cli:

  1. If there is a problem in installing angular-cli, use this code instead:
    sudo npm install --unsafe-perm -g @angular/cli
    reference: https://github.com/nodejs/node-gyp/issues/454
  2. [important] Use angular-cli to create a component:
    ng g c demo
